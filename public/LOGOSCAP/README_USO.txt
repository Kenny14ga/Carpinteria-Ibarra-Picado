Carpintería Ibarra Picado - assets individuales

Incluye SVG editables y PNG listos para usar:
- logo_principal: pantalla de login, landing y reportes.
- logo_horizontal_web: encabezado de la PWA / navbar.
- icono_pwa_1024: icono base de app.
- pwa_icon_192x192 y pwa_icon_512x512: tamaños recomendados para manifest.json.
- favicon_512: favicon / acceso directo.
- simbolo_solo: marca sin texto para botones, loader o avatar.
- logo_monocromatico: versión de una tinta para facturas, recibos o impresión.
- wordmark_sin_icono: nombre sin símbolo.
- paleta_colores: colores base.
